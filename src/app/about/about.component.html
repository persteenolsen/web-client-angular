Description

Angular SPA Client work together with a secure .NET Core 2.2 Web API serving as a Membership System of Users

Last Updated
  • 01-06-2023
Main functionality
  • It is possible to register and obtain an Account and perform a Login
  • A User can delete and update own Account after performing successfully login
  • An Admin can perform CRUD towards the Users after performing successfully login
  • A List of Posts ( CRUD ) from a DB after a successfully login
  • A List of Posts from a DB without doing a login
  • A simple demo about state management in Angular using RxJS
  • Cool Alert box popup for response on input values - success / errors
Module Bundler
  • Manuel configuration of Webpack 4 serving as a module bundler
Security
  • The Passwords of the Users are encrypted by BCRYPT before stored in the database
  • The login system is using a JSON Web Token ( JWT )
  • The user will have access to the secure endpoints of the Web API passing login and using the JWT
  • The endpoints of the Web API are secured by JWT
  • The Web API is secured by CORS allowing only the subdomain containing the SPA client making HTTPS request
The Angular Single Page Application - The frontend
  • Angular 8
  • TypeScript
  • Less
  • HTML and CSS
  • Mainly JavaScript ES5, ES6 and ES7
  • Traditionel Bootstrap 4 by CDN for the responsive design
The .NET Core 2.2 Web API - The backend
  • .NET Core 2.2
  • Mainly JavaScript ES5, ES6 and ES7
The Database
  • MS SQL as the database
Hosting
  • The Angular SPA client is hosted at a traditionel Webhotel on an IIS
  • The Web API is hosted at a traditional Webhotel on an IIS
Text Editor
  • Visual Studio Code as Text Editor
Testing
  • Test Driven Developement and Unit Testing all the way developing the system
